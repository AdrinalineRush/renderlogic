Local Application Quickstart

To run Parayu locally, clone the main workspace repository, run npm install, and start in development mode using npm start. Native bindings (like uiohook-napi and smart-whisper) compile automatically on systems with standard C++ build tools (Xcode CLI on macOS or Visual Studio Build Tools on Windows).

Hotkeys & Shortcuts Configuration

The default dictation hotkey is mapped to Ctrl + Space (or double-press Fn on Apple Silicon). You can configure hotkeys in the app settings panel which stores the raw keycodes in your local config.json under the application data directory.

Local Custom Dictionary

Improve word recognition for regional slang and technical terms (e.g. video formats, camera models). Create a file named custom-dictionary.json inside the application folder and add key-value pairs representing custom pronunciations and transcript outputs.

Help & FAQ

Support Center

Find resources, setup guides, and troubleshooting steps for RenderLogic software products.

How do I authorize accessibility permissions for Parayu on macOS?

Parayu uses local keyboard simulation to paste transcribed text directly. On macOS, this requires Accessibility permissions. Open **System Settings -> Privacy & Security -> Accessibility**, add **Parayu**, and toggle the switch to enabled.

Is my voice data shared with any external servers?

No. Both Parayu and RenderMagic Director run entirely on your local machine. All neural network inferences (whisper.cpp, on-device text cleanup, and audio-video compositing) happen on your local CPU or GPU. We do not run any telemetry servers.

How do I configure the dictionary replacement values?

Open the app UI, navigate to the **Dictionary** tab, and enter the misheard term and the correct target word. The local database will immediately intercept subsequent matching dictations and replace them seamlessly.
